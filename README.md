# 2026 Product Designer Research Report

This repository contains a Korean-language HTML research report on the 2026 product designer job market.

## Files

- `index.html` — product designer job market outlook page
- `study-abroad.html` — SVA / study abroad / NIW / financial simulation page
- `product-designer-outlook-2026.html` — original working HTML file
- `design-guide.md` — visual and component design guide
- `content-guide.md` — data and content update guide
- `project-context.md` — project context and decision history

## Notes

The report is split into two web tabs:

- Product designer outlook: market outlook, AI impact, US city-level salary data, layoffs, and career strategy
- Study abroad / SVA / NIW: graduate school options, Samsung vs SVA vs NIW decision framework, and financial simulation
