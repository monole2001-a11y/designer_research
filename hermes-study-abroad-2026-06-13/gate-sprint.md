## 0. 오늘 결론

현재 최우선은 **G0 입학 상태 확인**이다.

G0이 닫히지 않으면 다음 계산이 전부 흔들린다.

- Fall 2026 합격 + I-20 보유 → B(지금·국비·SVA) 계산 계속 진행.
- 합격/I-20 없음 → B는 사실상 Fall 2027로 재정의되고, C(2년 후·자비·탑급 HCI)와 시간 격차가 줄어든다.
- 합격은 있으나 I-20 없음 → I-20 발급 가능일과 CIP 표기 확인이 최우선.

## 4. G1/G2 병렬 준비

G0이 통과 또는 잠정 통과면 바로 아래를 진행한다.

### 4-0. 2026-06-13 14:30 KST 공식자료 확인 메모

- SVA MFA Interaction Design FAQ는 MFA IxD가 STEM program이고 CIP code가 `30.3101`, SEVIS listing이 `Human Computer Interaction`이라고 안내한다. 다만 G2는 웹페이지 문구만으로 닫지 말고 **본인 I-20의 CIP/major field + DSO 서면 회신**으로만 닫는다.
- USCIS는 F-1 OPT를 전공 관련 임시 고용허가로 설명하며, 일반 OPT는 최대 12개월이다. STEM OPT는 qualifying STEM degree 보유자가 post-completion OPT에 대해 **24개월 extension을 신청**하는 구조다. 즉 “자동 36개월”이 아니라 “12개월 OPT + 요건 충족 시 24개월 STEM OPT 신청 가능”으로 표현한다.
- DHS/Study in the States CPT 자료는 CPT가 DSO의 SEVIS authorization과 I-20 출력이 필요한 특정 employer/기간 기반 training이라고 설명한다. full-time CPT 1년은 OPT eligibility를 없앨 수 있으므로 재학 중 인턴십 가정은 학교 CPT policy 확인 전 보수적으로 둔다.
- DS-160은 CEAC에서 시작하며 application ID 기록, 20분 inactivity timeout, 약 90분 소요, 본인 전자서명이 핵심 주의사항이다.
- 상세 링크는 `objective-sources.md/html`에 분리했다.

### G1 — 국비 약관 원문 확보

확인해야 할 항목:
- 전형: 일반 $80K인지 꿈나래 $100K인지
- 지급 시점: 2026-08-01 납부 일정과 맞는지
- 생활비 사용 가능 여부
- 유지 조건: 성적, 재학 상태, 휴학/자퇴 시 반환
- 반환 조건: 언제, 얼마, 어떤 사유에서 반환인지
- 결과보고/의무사항 불이행 페널티
- 졸업 후 귀국 의무 없음이 원문 또는 서면으로 확인되는지

보낼 질문 초안:

```text
안녕하세요. 장학금 수혜 조건 확인을 위해 문의드립니다.

1. 제 전형의 총 지원금은 2년 기준 $80,000인지 $100,000인지 확인 부탁드립니다.
2. 2026 Fall 등록금 납부일 전에 1차 지급이 가능한지, 지급 예정일을 확인 부탁드립니다.
3. 지원금을 등록금 외 생활비/보험료/학과비 등에 사용할 수 있는지 확인 부탁드립니다.
4. 성적, 재학 유지, 휴학/자퇴, 중도 포기 시 반환 조건이 어떻게 되는지 원문 기준으로 안내 부탁드립니다.
5. 졸업 후 귀국 의무 또는 특정 국가/기관 근무 의무가 있는지 확인 부탁드립니다.
6. 위 조건이 정리된 약관/공고/계약서 원문을 받을 수 있을까요?

감사합니다.
```

## 6. 다음 업데이트 방식

사용자가 G0 요약을 주면 Hermes는 아래 중 하나로 판정한다.

- `G0 통과`: Fall 2026 합격 + I-20 보유가 확인됨.
- `G0 조건부 통과`: 합격은 확인, I-20 발급 일정/요건 확인 필요.
- `G0 미완료`: 합격 또는 I-20 증거가 부족함.
- `G0 불통과`: Fall 2026 B(지금·국비·SVA)가 성립하지 않음.

판정 후 바로 다음 액션은 G1 국비 약관 원문, G2 I-20 CIP/STEM 확인, G3 재정 스트레스 테스트 순서로 간다.
